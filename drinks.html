When gradually reducing the size the drink names (col-md) is likely to break first. This is because the letters row (col-sm-2) is configured to adjust at smaller screen sizes (sm), while the drink names row (col-md) is set to adapt at medium screen sizes (md). Consequently, the drink names row will merge before the letters row as the screen size decreases.
